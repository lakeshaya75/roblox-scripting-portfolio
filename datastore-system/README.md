
# DataStore Framework

## Overview

A reusable Roblox DataStore framework responsible for loading, saving, and managing persistent player data.

## Features

- Automatic player loading
- Automatic saving
- Error handling
- Session management
- Modular architecture
- Easy expansion

## Technologies

- DataStoreService
- RemoteFunction
- RemoteEvent
- HttpService
- Player lifecycle events
- UpdateAsync
- Autosave loop
- Server-side validation
