# Inventory & Crafting System

## Overview

A client-server inventory and crafting system for a Roblox survival game. Players can collect harvested resources, view items in an inventory UI, open/close animated menus, craft tools using required materials, and receive crafted tools from the server.

## Features

- Inventory UI
- Crafting UI
- Resource stacking
- Item count display
- Item icons
- Item descriptions
- Animated menu transitions
- Button hover effects
- Material requirement checks
- Crafting feedback messages
- Server-side tool delivery
- Harvest reward integration

## Technologies

- Luau
- RemoteEvents
- ScreenGui UI
- TweenPosition
- TweenSize
- ServerStorage
- ReplicatedStorage
- Client-server architecture
