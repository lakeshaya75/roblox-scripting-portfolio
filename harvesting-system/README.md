# Harvesting System

## Overview

A client-server harvesting system for Roblox survival games. Players can equip tools, swing/hold to harvest nearby resources, play animations and sounds, damage harvestable objects, and trigger resource removal/respawning when depleted.

## Features

- Tool-based harvesting
- Hold-to-harvest behavior
- Client-side animation playback
- Hitbox detection
- Server-side cooldowns
- Server-side resource validation
- Tool damage attributes
- Resource health attributes
- Hit sounds
- Tree shake feedback
- Harvest reward event
- Resource respawn integration

## Technologies

- Luau
- RemoteEvents
- Tool objects
- AnimationTracks
- Workspace:GetPartBoundsInBox
- Attributes
- Model pivoting
- Task scheduling
- Client-server architecture
