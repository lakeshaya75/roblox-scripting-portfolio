# Procedural Resource Spawning System

## Overview

A server-side procedural resource spawning system built for a Roblox survival game. The system places harvestable resources across different biome regions using terrain raycasting, spacing rules, weighted resource distribution, and player-aware respawning.

## Features

- Biome-based resource spawning
- Configurable region rules
- Terrain raycasting
- Material-based spawn filtering
- Slope validation
- Global spacing between resources
- Same-type spacing rules
- Weighted resource balancing
- Randomized resource rotation
- Ground offset support
- Player-aware respawning
- Server-side resource tracking
- Integration with harvesting systems

## Technologies

- Luau
- ServerScriptService
- ServerStorage
- Workspace Raycasting
- RaycastParams
- Terrain Material Detection
- Attributes
- Global server functions
- Task scheduling
