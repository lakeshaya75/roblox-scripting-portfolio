# Story & Event Framework

## Overview

A server-driven event framework for Roblox story games that manages cutscenes, dialogue, objectives, NPC movement, player teleportation, environment changes, and scripted gameplay sequences.

## Features

- Story progression
- Cutscene orchestration
- Dialogue system integration
- NPC movement
- Objective system
- Timers
- Camera shake events
- Environment manipulation
- Dynamic lighting changes
- Player teleportation
- Item distribution
- Multiplayer synchronization

## Technologies

- Luau
- RemoteEvents
- Client-server architecture
- Tween/Camera systems
- NPC pathing
- Workspace manipulation
